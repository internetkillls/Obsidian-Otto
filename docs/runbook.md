# Runbook

## First install

1. unzip repo
2. run `initial.bat`
3. choose vault path or the bundled sample vault
4. choose whether to enable Docker helpers
5. wait for Bronze → Silver → Gold pipeline
6. open `tui.bat`

## Everyday commands

### status
```bat
status.bat
```

### TUI
```bat
tui.bat
```

### run a fresh pipeline
```bat
reindex.bat
```

### one-shot KAIROS
```bat
kairos.bat
```

### one-shot Dreaming
```bat
dream.bat
```

### sync OpenClaw live config
```bat
sync-openclaw.bat
```

### clean Docker
```bat
docker-clean.bat
```
