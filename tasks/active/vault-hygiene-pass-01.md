# Task: vault-hygiene-pass-01

## Goal
Identify the highest-risk folders in the vault and determine whether they need metadata repair, sorting, or a deeper restructuring pass.

## Scope
- Start with a small subset of folders.
- Prefer scoped refreshes over full-vault rebuilds.

## Required evidence
- folder severity score
- missing frontmatter count
- duplicate note title count
- broken or suspicious wikilink count
- three representative files if risk is high
